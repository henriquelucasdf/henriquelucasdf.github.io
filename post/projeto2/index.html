Introdução

Este projeto consiste em uma previsão de vendas para o mês de dezembro de uma loja do ramo de bijuterias localizada no Distrito Federal. Trata-se de uma franqueada que está há 10 anos no mercado, com um market share consolidado e que já empregou dezenas de pessoas.

No final de outubro de 2020, com o cenário turbulento da pandemia do novo coronavírus, fui procurado para realizar uma previsão das vendas para esta loja além de uma análise exploratória dos dados. A empresa ficou algumas semanas fechada devido à quarentena, o que resultou em um gap nos dados, além de uma queda abrupta no faturamento. Apesar do crescimento mês a mês nas vendas após a reabertura, o cenário de incerteza ainda prevalece, com a possibilidade de uma segunda onda do vírus no Distrito Federal e a falta de confiança dos clietes.

Optou-se por utilizar a abordagem de previsão via séries temporais, já que o conjunto de dados apresenta uma alta sazonalidade. O modelo Prophet foi utilizado por possuir uma curva de aprendizado curta, ser altamente flexível, possuir suporte para a inserção de feriados, lidar bem com dados faltantes e possuir diversas funcionalidade embutidas na própria biblioteca.

Destaca-se ainda que, como a empresa renova seu estoque antecipadamente para o natal, foi-me disponibilizado apenas uma semana para realizar todo este projeto, o que se mostrou um verdadeiro desafio.
